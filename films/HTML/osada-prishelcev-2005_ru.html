Осада пришельцев

В фильме «Осада пришельцев» (Alien Siege)представители высокоразвитой внеземной цивилизации кулку захватывает Землю для удовлетворения своих потребностей по спасению собственной расы от генетического заболевания. Человеческая кровь является основным материалом для изготовления лечебной сыворотки для инопланетян. Восемь миллионов землян должны отдать свои жизни для восстановления популяции кулку. Для доктора Чейза - это потеря горячо любимой дочери, и на такую потерю он не согласен. Среди землян находятся еще люди не согласные отдать жизнь для спасения инопланетной цивилизации, они объединяются для противостояния.
