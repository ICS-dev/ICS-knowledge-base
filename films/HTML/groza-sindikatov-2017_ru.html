Гроза синдикатов

В представленной картине пойдет речь об опытных бойцах, которые встревают в разборки с мафией из разных стран. Они устраняют цели по заказу: сначала русских, потом свидетеля по делу с обвинением Рассела Дипполито. Однако Крюков, для которого герои выполняли последний заказ, отказался выполнять свои обязательства. Русские планируют развязать войну с итальянской мафией на Западном Побережье, чтобы расширить сферу влияния. И отряд наемников занимает в этом плане одну из ключевых позиций. Вот только герои понимают, что, как только работа будет выполнена, их устранят, как ненужных свидетелей.
